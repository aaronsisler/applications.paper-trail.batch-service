# Paper Trail Batch Service

## Development

### Run

To run locally on [localhost:8080](http://localhost:8080):

``` bash
gradle run
```

### Test

To run unit tests:

``` bash
gradle test
```

To view the test report:

```bash
open build/reports/tests/test/index.html
```
